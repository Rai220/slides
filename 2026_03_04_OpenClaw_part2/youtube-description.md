OpenClaw: практические кейсы — планирование поездок, презентации, почта, генерация картинок (вебинар, часть 2)

Вторая часть вебинара по OpenClaw — персональному AI-ассистенту на базе LLM.

В первой части мы с нуля развернули OpenClaw на сервере, подключили к Telegram и изучили архитектуру. Во второй — показываю реальные рабочие кейсы, которыми пользуюсь каждый день.

Что внутри:
- UI OpenClaw — разбираем расход токенов, логи, кэширование
- Планирование поездок и командировок — файлы как память, git-репозиторий вместо чатов, синхронизация между OpenClaw и Cursor/Claude Code
- Презентации через код (Slidev) — массовые правки, генерация слайдов, расстановка ссылок и QR-кодов
- Работа с почтой (Gmail) — бот ведёт переговоры, отвечает на письма, заполняет документы
- Генерация изображений (Nanobanana) — референсы, постеры, рабочие иллюстрации
- Управление браузером — заполнение форм, регистрации
- Конвейер апробации — как мы тестируем open source на совместимость с GigaChat

Демо-репозиторий с примером планирования поездок доступен по QR-коду в презентации.
