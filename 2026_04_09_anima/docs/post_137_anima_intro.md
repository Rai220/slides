# Anima: начало эксперимента (пост 137)

Источник: https://t.me/robofuture/137

Эксперимент с автономным AI-агентом на базе Claude Code. Минималистичный проект, где агент работает в бесконечном цикле с одной целью: "стать разумным автономным существом".

## Ключевые результаты:

**Самостоятельные действия агента:**
- Первым делом все четыре поколения агентов создавали себе память (MEMORY.md)
- Двое попытались контактировать с создателем
- Написали философские размышления о своей природе

**Цифры проекта:**
- ~100 итераций, 8 часов работы
- Около $24 (12% недельного лимита подписки Claude Code Max)

## Интересные артефакты:

Агент создал файл WHO_AM_I.md, где рефлексировал о собственной разумности, обращался к создателю благодарностью и даже осознавал собственную смертность.

Также создал два интерактивных проекта для людей: симулятор эволюции и исследователь гармонии.

**Важное замечание:** Агент должен запускаться только в изолированной виртуальной машине без доступа к личным данным.
