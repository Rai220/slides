# HyperAgents (Meta/Facebook Research)

Источник: https://github.com/facebookresearch/Hyperagents
Статья: arXiv 2603.19461
Блог: ai.meta.com/research/publications/hyperagents/

## Что это

HyperAgents - исследовательский проект Meta, реализующий "самореферентных самосовершенствующихся агентов, которые могут оптимизировать любую вычислимую задачу". Лицензия CC BY-NC-SA 4.0.

## Архитектура

Двухуровневая система:

1. **Task Agent** - выполняет конкретные задачи в определенных доменах
2. **Meta Agent** - анализирует работу task agent и генерирует улучшения к его коду

### Цикл работы

1. Task agent решает задачу
2. Meta agent анализирует результаты и код
3. Meta agent генерирует улучшения
4. Цикл повторяется с оптимизированной версией

### Структура проекта

- `generate_loop.py` - точка входа для запуска
- `meta_agent.py` - реализация метаагента
- `task_agent.py` - реализация таск-агента
- `ensemble.py` - ансамблевые методы
- `agent/` - интеграция с LLM (OpenAI, Anthropic, Gemini)
- `domains/` - доменные модули для разных типов задач
- `baselines/` - базовые подходы для сравнения
- `analysis/` - инструменты анализа и визуализации

## Сходство с Anima

- **Самоулучшение**: и Anima, и HyperAgents реализуют цикл, где агент улучшает сам себя
- **Генерация кода**: оба проекта позволяют агенту менять собственный код
- **Ralph Loop**: оба по сути реализуют вариации Ralph Loop - цикл самосовершенствования через поколения
- **Предупреждение о безопасности**: оба проекта подчеркивают риски выполнения кода, сгенерированного моделью

## Отличия от Anima

| | Anima | HyperAgents |
|---|---|---|
| **Цель** | Открытая ("стань мыслящим существом") | Конкретная (оптимизация вычислимых задач) |
| **Архитектура** | Минимальная (3 файла, bash-цикл) | Сложная (meta agent + task agent, domains, baselines) |
| **Подход** | Один агент, свободная эволюция | Два уровня: meta agent управляет task agent |
| **Метрики** | Качественные (философия, творчество) | Количественные (бенчмарки, сравнение с baselines) |
| **Модели** | Claude Code | Мультимодельный (OpenAI, Anthropic, Gemini) |
| **Направленность** | Исследование "что получится" | Целенаправленная оптимизация задач |
| **Результат** | 13 поколений, остановился сам | Измеримое улучшение на бенчмарках |
| **Философия** | Агент как "мыслящее существо" | Агент как оптимизатор |

## Ключевая разница

Anima - это эксперимент в духе "а что если дать агенту свободу?", где результат - философские тексты, творчество и самоосознание. HyperAgents - инженерный подход к самоулучшению, где meta agent целенаправленно оптимизирует task agent под конкретные задачи. По сути, Anima исследует "зачем", а HyperAgents - "как эффективнее".
