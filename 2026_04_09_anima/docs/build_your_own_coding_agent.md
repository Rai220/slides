# Build Your Own Coding Agent — обучающий репозиторий

**Репозиторий:** https://github.com/labdmitriy/build-your-own-coding-agent
**Автор:** Дмитрий Лабазкин (внутренний вебинар, репозиторий в открытом доступе)
**Источник:** https://t.me/gigatrash/209

## Что это?

Обучающий репозиторий по сборке coding-агента с нуля — без фреймворков, только Python и Pydantic. 13 шагов, в каждом одна новая концепция. К шагу 6 агент уже умеет редактировать собственный код и подхватывать изменения без перезапуска.

## Основа

За основу взяты материалы **Ивана Лео** — бывшего инженера Manus (сейчас в Google DeepMind), написавшего серию статей "Openclawd From Scratch". Это разбор минималистичного агентного цикла внутри OpenClaw.

## Связь с Ralph Loop

Репозиторий учит строить тот самый **inner loop** (agent loop / tool loop) — цикл read → evaluate → act, который является ядром любого coding-агента и, соответственно, ядром Ralph Loop. Понимание того, как устроен inner loop, помогает глубже понять, почему Ralph Loop работает: bash-цикл (outer loop) просто перезапускает этот inner loop со свежим контекстом.

## Ключевая ценность

Материал помогает понять, как строить агентов, даже если вы взаимодействуете с ними на верхнем уровне (через CLI/harness). Понимание внутренней механики делает вас лучшим "инженером среды" для Ralph Loop.
